# Wave Solutions Staging Asset Prompts

Use these prompts for the local staging branch only. Keep the output photographic with subtle compositing, soft technical atmosphere, restrained UI overlays, and Wave brand colors leaning navy, white, and orange. Avoid exaggerated futuristic dashboards, purple bias, and heavy stock-photo styling.

## General Rules

- Aspect ratios should match the intended placement below.
- Real people should look professional, warm, and credible.
- Background overlays should be subtle phone, email, calendar, message, and network motifs rather than large interface mockups.
- Lighting should stay bright, calm, and service-business appropriate.
- Every image should feel part of one set.
